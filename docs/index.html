ドコカワ

STEPの設変、どこ変わったかわからない問題。
自分が困ってたので作りました。
ダウンロード（Windows / 無料）
インストール不要 · ネット接続不要 · ZIP展開するだけ

こういう状況で使ってます

  • 「修正しました」とだけ書いてあるSTEPが戻ってきた。どこを？
  • CADで両方開いて部品を一個ずつ見比べる。50部品あると無理
  • CADの比較機能、STEPには対応してない
  • 結局目視。見落としが怖いまま次工程に流す
STEPを2つ放り込むと、変更・追加・削除された部品が一覧で出ます。

何ができるか

STEP差分チェック

変更・追加・削除を一覧表示。BBox差、体積差、面数の変化も出る。これがメイン機能。

3Dビューア

差分が出た部品をクリックするとブラウザ上で3D表示。回転・ズームできる。

ローカル完結

外部通信なし。STEPは処理後に消える。社外秘でも安心して使える。

CSV / Excel出力

差分結果をExcelに出せる。設変記録としてそのまま使える。

他にDfM警告（薄壁・深穴の自動検出）、仕様チェック（STEPとPDFの数値突き合わせ）もあります。作り始めたら増えた。

画面イメージ

変更・追加・削除がテーブルで一覧表示。変更前後の形状を3Dで比較できます

上面図・側面図など複数の視点で変更箇所を確認

動作環境・仕様

ネット接続 不要（完全ローカル動作）
対応形式 STEP（.stp / .step）、PDF（仕様チェック機能）
